What is WebGL?

WebGL (Web Graphics Library) is a JavaScript API for rendering high-performance interactive 3D and 2D graphics within any compatible web browser without the use of plug-ins

All modern browsers support WebGL

WebGL allows developers to create GPU-accelerated graphics that can be accessed across different devices without requiring installation of additional software or plugins.

WebGL is based on OpenGL ES, which is a subset of OpenGL designed for embedded systems like mobile devices. WebGL closely follows OpenGL ES specifications.

The process begins with the specification of geometric data (vertices) that define the shapes to be rendered.

Each vertex undergoes processing through the vertex shader. This shader can transform the vertex position, calculate lighting effects, or perform other operations on a per-vertex basis.

Tessellation subdivide a shape further using a tessellation shader

The geometry shader can further process primitives. This shader can create new vertices, generate additional primitives, or perform other geometry-related tasks.

The vertices are assembled into geometric primitives such as points, lines, or triangles, based on the specified drawing mode.

Rasterization: Each primitive (triangle) is rasterized into fragments (pixels). This involves determining which pixels are covered by the primitive.

Fragment Shader: Fragments undergo processing in the fragment shader. This shader computes the final color of each fragment, applying lighting, textures, and other effects.

Depth Test: Fragments undergo depth testing to determine visibility based on the depth buffer.

Stencil Test: Optional test to control writing to the stencil buffer.
